# Dot files for all configs

## NB

this is currently in testing

each folder is indepent and doesnt depend on any

## INSTALL

go into any of the folders and install

```sh
cd <folder>
chmod +x install.sh 
./install.sh
```

## FOLDERS

- `bspwm`

contains bspwm files

- `i3`

contains i3 files

- `i3-vm`

contains i3 files but made for virtual machines ( smaller, less functionality )
